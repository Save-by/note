# 汇编命令

## 寄存器

不同位数的通用寄存器

| 32位(0-FFFFFFFF) | 16位(0-FFFF) | 8位(0-FF) |
| :--------------: | :----------: | :-------: |
|       EAX        |      AX      |    AL     |
|       ECX        |      CX      |    CL     |
|       EDX        |      DX      |    DL     |
|       EBX        |      BX      |    BL     |
|       ESP        |      SP      |    AH     |
|       EBP        |      BP      |    CH     |
|       ESI        |      SI      |    DH     |
|       EDI        |      DI      |    BH     |

8位：L表示低8位，H表示高8位



### 指令

1. 向寄存器存值

   ```shell
   mov 目标地址,存的数值
   mov 目标地址,源地址
   ```
