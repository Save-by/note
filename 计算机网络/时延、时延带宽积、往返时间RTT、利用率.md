# 时延、时延带宽积、往返时间RTT、利用率

## 时延

指数据（报文/分组/比特流）从网络（或链路）的一端传输到另一端所需的时间。也叫**延迟**或**迟延**。单位为s。

时延分类：

1. 发送时延（传输时延）
   - 发送时延 = 数据长度/信道带宽（发送速率）
2. 传播时延
   - 传播时延 = 信道长度/电磁波在信道上传播速率
3. 排队时延（等待输出/输入链路可用）
   - 排队时延 = 等待输出/输入链路可用所用的时间
4. 处理时延（检错、找出口）
   - 处理时延 = 检错、找出口所用的时间



## 时延带宽积

1. 时延带宽积 = 传播时延*带宽 ， 单位是bit
2. 时延带宽积又称为以**比特为单位的链路长度**。即“某段链路现在有多少比特”。
3. 用来表示容量



## 往返时间RTT

1. 从发送方发送数据开始，到发送方收到接收方的确认（接收方收到数据后立即发送确认），总共经历的时延。
2. RTT越大，在收到确认之前，可以发送的数据越多。
3. RTT包括
   1. 往返传播时延 = 传播 * 2
   2. 末端处理时间





## 利用率

1. 信道利用率
   - 信道利用率 = 有数据通过时间/（有 + 无）数据通过时间
2. 网络利用率
   - 网络利用率 = 信道利用率加权平均值
