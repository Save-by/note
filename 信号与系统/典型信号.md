# 典型信号

**信号的表示**

- 函数表达式f(t)
- 波形

## 几种典型确定性信号

### 指数信号

f(t) = Ke^αt^

- α = 0 直流信号（常数）
- α < 0 衰减信号
- α > 0 增幅信号
