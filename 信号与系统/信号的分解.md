# 信号的分解

为了便于研究信号的传输和处理问题，往往将信号分解为一些简单（基本）的信号之和，分解角度不同，可以分解为不同的分量

1. 直流分量与交流分量
2. 偶分量与奇分量
3. 脉冲分量
4. 实部分量与虚部分量
5. 正交函数分量

## 正交函数分量

如果用正交函数集来表示一个信号，那么，组成信号的各分量就是相关正交的。把信号分解为正交函数分量的研究方法在信号与系统理论中占有重要地位。
